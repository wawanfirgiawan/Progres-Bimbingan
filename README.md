# AutoRumpon Progress Tracker ✅

**AutoRumpon** adalah sistem pelacakan progres berbasis web yang dibuat untuk memantau perkembangan kegiatan proyek *Inovasi Mesin Pemutar Otomatis* dalam mendukung produktivitas kelompok usaha tali rumpon di **Desa Karama, Sulawesi Barat**.

Website ini memungkinkan setiap anggota tim untuk mencentang tugas yang telah selesai dan memvisualisasikan progres secara real-time melalui progress bar. Data progres tersimpan otomatis di browser pengguna dengan `localStorage`.

---

## 🔧 Fitur Utama

- ✅ Checklist tugas-tugas tim lengkap dengan nama anggota
- 📊 Progress bar otomatis ter-update
- 🔒 Penyimpanan lokal tanpa backend (`localStorage`)
- 🔁 Tombol Reset Progres
- 📥 Ekspor progres ke file `.txt`
- 🖨️ Cetak laporan progres dari browser

---

## 📁 Struktur File

```bash
autorumpon-progress-tracker/
│
├── index.html       # Halaman utama
├── style.css        # Gaya dan tampilan website
└── script.js        # Logika dan penyimpanan progres
````

---

## 🚀 Cara Menjalankan

1. Clone repository ini atau unduh ZIP-nya.
2. Buka file `index.html` menggunakan browser (Chrome, Firefox, dll).
3. Centang progres dan lihat progress bar berjalan otomatis.
4. Gunakan tombol **Reset**, **Export**, atau **Cetak** sesuai kebutuhan.
